go-hyperestraier-win32
======================

This is the dll-wrapper of HyperEstraier for the programming language Go.

* It supports GOOS=windows and GOARCH=386.
* It requires these libraries.
    * estraier.dll
    * libgnurx-0.dll
    * libiconv-2.dll
    * pthreadGC2.dll
    * qdbm.dll
    * zlib.dll
    * zlib1.dll
